두 가지 플레이 모드

공격 모드 ▶

  • 준비 화면: 유닛 구매 + 마법 선택
  • 특수 건물 패널 비표시
  • 아군은 좌측에서 출발
  • 안개(FOW) 적용 — 시야 확보 필요
  • 적 기지는 5선 방어 구조
  • 시간 제한 180초

수비 모드 🛡

  • 준비 화면: 유닛 배치 + 특수 건물 업그레이드
  • 마법 섹션 비표시
  • 아군 기지는 좌측(-10 성벽 라인)
  • FOW 없음 — 전장 전체 가시
  • 5파 웨이브를 방어
  • 성벽 클릭으로 문 위치 변경 가능

성벽 슬라이드 이동

유닛은 물리 기반이 아닌 transform.position 직접 이동. SphereCast로 수직 장애물 감지 → 법선 벡터에 투영해 슬라이드. 문(Gate) 구간은 콜라이더 없음 → 자유 통과.

🤖 12. AI 시스템

TestSimpleUnitAI (유닛 AI)

타깃 우선순위

1. 명령 타깃 (CommandAttack)
2. 위협 범위 내 적 유닛 (nearest)
3. 가장 가까운 적 건물

이동

transform.position 직접 이동
SphereCast → 수직 장애물 슬라이드
0.5초마다 타깃 재탐색

버프 슬롯

DamageMultiplier: 기본 1.0 (분노 마법으로 일시 증가)
SpeedMultiplier: 기본 1.0 (빙결 마법으로 일시 감소)

건물 AI (수비 자동)

컴포넌트동작
TestTowerAI 사거리 내 적 유닛 자동 조준 · 공격. isPlayerSide 플래그로 아군/적 구분.
TestMageTowerAI 사거리 14m, 데미지 35, 쿨다운 2.5s. 발사체 2개 동시 발사.
TestBuffBuildingAI 주변 아군 유닛에게 주기적으로 DamageMultiplier / SpeedMultiplier 버프 적용.
건물 AI는 항상 자동으로 동작. 공격 모드에서 플레이어가 건물을 직접 조작할 필요 없음.
